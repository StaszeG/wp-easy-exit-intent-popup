FREE, easy to use plugin that shows an exit intent popup

== Description ==
FREE, super simple and quick to use, plugin that shows an exit intent popup with an uploaded image. Doesn\'t require creating any external accounts.

After plugin installation and activation new menu option will be available
Settings > Exit intent popup

Here you will have to upload the image.
Now the image will be showed as a exit intent popup on your wordpress page!
